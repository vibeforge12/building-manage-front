# 유저 타입별 기능 명세

## 1. 유저 (일반 거주자)

### 주요 기능
- **대시보드**: 개인 정보, 공지사항, 급여/관리비 확인
- **신고/요청**: 시설 고장 신고, 서비스 요청
- **커뮤니티**: 공지사항 확인, 이웃과의 소통
- **예약**: 공용시설 예약 (헬스장, 회의실 등)
- **결제**: 관리비 결제, 결제 내역 확인

### 권한
- 자신의 정보만 조회/수정
- 신고 및 요청 등록
- 공용시설 예약
- 공지사항 조회

## 2. 관리자 (건물 관리자)

### 주요 기능
- **대시보드**: 건물 전체 현황, 긴급 상황 모니터링
- **거주자 관리**: 입주자 정보 관리, 계약 관리
- **시설 관리**: 시설 점검, 유지보수 스케줄
- **신고 처리**: 거주자 신고 접수 및 처리
- **공지사항**: 공지사항 작성 및 발송
- **예약 관리**: 공용시설 예약 승인/거부
- **재정 관리**: 관리비 관리, 예산 관리

### 권한
- 건물 내 모든 정보 조회
- 거주자 정보 관리
- 신고 처리 및 담당자 배정
- 공지사항 작성/수정/삭제
- 예약 승인/거부

## 3. 담당자 (유지보수 담당자)

### 주요 기능
- **작업 대시보드**: 배정된 작업 목록, 진행 상황
- **작업 처리**: 신고된 문제 해결, 작업 완료 보고
- **스케줄 관리**: 정기 점검 스케줄, 작업 일정
- **자재 관리**: 필요 자재 요청, 사용 내역 기록
- **보고서**: 작업 완료 보고서 작성

### 권한
- 배정된 작업만 조회/처리
- 작업 상태 업데이트
- 자재 요청
- 작업 보고서 작성

## 4. 본사 (관리 본사)

### 주요 기능
- **전체 대시보드**: 모든 건물 통합 현황
- **건물별 분석**: 각 건물 운영 현황 분석
- **관리자 관리**: 각 건물 관리자 관리 및 평가
- **담당자 관리**: 유지보수 담당자 배정 및 관리
- **재정 분석**: 전체 수익/비용 분석, 예산 관리
- **성과 관리**: KPI 모니터링, 성과 평가
- **정책 관리**: 전사 정책 수립 및 배포

### 권한
- 모든 건물 데이터 조회
- 관리자 및 담당자 계정 관리
- 전사 정책 설정
- 재정 데이터 분석
- 시스템 설정 관리

## 공통 기능

### 모든 유저 타입
- **알림**: 푸시 알림, 인앱 알림
- **프로필**: 개인 정보 수정
- **설정**: 앱 설정, 알림 설정
- **도움말**: FAQ, 고객 지원
