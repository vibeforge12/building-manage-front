# 엄지(UMJI) - 건물 관리 앱 랜딩 페이지 기획서

## 1. 개요

### 1.1 프로젝트 정보
- **서비스명**: 엄지 (UMJI)
- **서비스 유형**: 다중 역할 기반 건물 관리 모바일 애플리케이션
- **운영사**: (주)메라키플레이스
- **목표**: 앱 스토어 배포를 위한 서비스 소개 랜딩 페이지

### 1.2 랜딩 페이지 목적
1. 서비스 소개 및 주요 기능 안내
2. 앱 다운로드 유도 (App Store / Google Play)
3. 법적 필수 정보 제공 (이용약관, 개인정보처리방침 등)
4. 기업/광고 제휴 문의 안내

### 3.5 FAQ 섹션 (자주 묻는 질문)

**목적**: 사용자 의문 해소 및 신뢰 구축

**예상 Q&A**:

```
Q. 엄지는 어떤 서비스인가요?
A. 엄지는 입주민, 관리자, 담당자, 본사를 연결하는 스마트 건물 관리 플랫폼입니다.

Q. 회원가입은 어떻게 하나요?
A. 앱에서 건물, 동/호수 정보를 입력하여 가입 신청하면, 관리자 승인 후 이용 가능합니다.

Q. 민원은 어떻게 접수하나요?
A. 메인 화면에서 부서를 선택하고, 민원 내용을 작성하여 접수하면 됩니다.

Q. 담당자가 없으면 민원 접수가 안 되나요?
A. 해당 부서에 담당자가 배정되어 있고, 출근한 상태일 때 민원 접수가 가능합니다.

Q. 개인정보는 안전하게 보호되나요?
A. 네, 암호화된 보안 저장소에 안전하게 보관되며, 개인정보처리방침에 따라 관리됩니다.
```

## 5. 법적 문서 페이지

### 5.1 이용약관 (/terms)
- 기존 `legal_documents.dart`의 `termsOfService` 내용 사용
- [회사명] → (주)메라키플레이스로 치환
- 연락처 정보 업데이트

### 5.2 개인정보처리방침 (/privacy)
- 기존 `legal_documents.dart`의 `privacyPolicy` 내용 사용
- [회사명], [담당자명], [연락처] 등 플레이스홀더 치환
- 개인정보 보호책임자 정보 업데이트

### 5.3 위치기반서비스이용약관 (/location)
- 새로 작성 필요 (위치 기반 기능 사용 시)

## 6. 기술 스택 제안

### 6.1 프론트엔드 옵션

**Option A: 정적 사이트 (추천)**
- Next.js + Tailwind CSS
- Vercel 배포
- 장점: SEO 최적화, 빠른 로딩, 간편한 배포

**Option B: 심플 HTML**
- HTML + CSS + JavaScript
- GitHub Pages / Netlify 배포
- 장점: 가장 간단, 유지보수 용이

**Option C: Flutter Web**
- 기존 프로젝트 활용
- 장점: 코드 재사용, 일관된 디자인

### 6.2 도메인 & 호스팅
- 도메인: umji.app 또는 umji.kr (제안)
- SSL 인증서: Let's Encrypt (무료)
- CDN: Cloudflare (무료 티어)
